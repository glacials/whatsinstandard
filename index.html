What is Standard?

Standard is a rotating group of Magic: The Gathering sets. Most sets enter Standard when they're released and drop out about eighteen months later. Reprint/Masters sets are Modern sets, not Standard sets. 1

A Standard card is a card printed or reprinted into a set currently in Standard.

A deck which contains only Standard cards is called a Standard deck.

Recent updates

If you're coming back to Magic after a break, note that the way Wizards organizes blocks has recently changed. Previously, each 12-month period saw one new core set and one new three-set block; today, each 12-month period sees instead two new two-set blocks (core sets are gone).2
